UnitCache<T> class

Unità della cache utile per manipolare i vari elementi in unità all'interno della cache. Questa unità ha come attributi un nome identificativo, un'icona, una data per capire quando è stato memorizzato un elemento nel tempo e aggiornarlo in base alle strategie di memorizzazione e l'elemento che è di un tipo generico T così da rendere tale unità slegata da ogni tipo di elemento memorizzato rendendola modulare.
